# 개요
백엔드 애플리케이션 개발 온-보딩 문서에 따라 다음 기능을 개발한다. 

참여자: 엄다빈, 최서우

## 기능
나만의 00 리스트 만들기
 - 맛집
 - 카페

## 협업 워크 플로우
1. git 레포지토리에 main  브랜치 공유
2. 각 기능마다 branch 만들어 기능 구현
3. 완성 후 main 브랜치에 merge

## 기타
- 학습한 내용에 대해서는 /NOTEBOOKS에 md 파일로 작성한다. 
- /docs 브랜치에서 관리한다.
